プロジェクト

画像生成AIによるバナー生成（2024/08 ~ 現在）

画像生成AI「StableDiffusion」を活用した広告バナー生成プロジェクトに従事。
InpaintingやControlNet、Img2Img、Tokenの上限突破、Regional Prompterなどの技術を駆使し、高品質なバナー画像を生成。
加えて画像生成のためのワークフローを構築。

効果検証（2023/03 ~ 現在）

クライアントの課題把握から施策結果のプレゼンまで一貫して担当。
基本属性ごとの集計分析に加え、線形回帰やSHAP値を用いた解釈性の高い分析を実施。ロジスティック回帰による傾向スコア算出とそのスコアを用いたIPWを実施し施策効果の定量化も行う。

マーケティングリストの作成（2023/03 ~ 現在）

基本的な抽出手法（例: 既契約者の除外）から、LightGBMによる予測確率を活用したターゲティングを実施。
また、バイアス除去のため、傾向スコアを使った重み付け学習によるターゲット選定を実施。
